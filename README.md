# Shopify - Full Stack E-Commerce Application

Shopify is a full-stack e-commerce web application built using Spring Boot and MySQL on the backend and HTML, CSS, and JavaScript on the frontend. The system supports product management, user handling, cart functionality, and order processing using REST APIs.

---

## Tech Stack

Backend:
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API

---

## Features

- Product listing from database
- Add to cart functionality
- Order placement system
- User registration and login
- REST API integration
- Layered architecture (Controller → Service → Repository)
- JPA entity relationships (User → Orders → OrderItems)
